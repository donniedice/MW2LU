This is an addon that will play the "LEVEL UP!" chime from Modern Warfare 2 when you level up in WoW.

------------------------------

- This addon works in DRAGONFLIGHT!

- This addon works in CLASSIC WOW!

- This addon works in CLASSIC WotLK!
